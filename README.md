# MentalHealth Connect: A Digital Assessment Tool

## 🌟 Overview
MentalHealth Connect is a web application designed to bridge the gap between mental health awareness and professional support. Built as part of a Project-Based Learning (PBL) course, this tool provides a safe and private space for individuals to evaluate their mental well-being using clinically validated assessment tools.

## 🎯 Key Features
* **Secure Patient Portal:** A private login system to ensure that an individual's mental health data remains confidential and accessible only to them.
* **Guided Self-Assessments:** Includes the GAD-7 (Anxiety) and PSS-10 (Stress) quizzes to help users understand their current emotional state.
* **Smart Support System (Triage):**
    * **Low Severity:** Users receive personalized wellness tips and mindfulness resources.
    * **High Severity:** The app immediately provides contact details for the National NIMHANS Helpline and verified professional counselors.
* **Educational Hub:** A home page dedicated to mental health awareness and destigmatizing the search for help.

## 🛠️ How it Works (Simple Overview)
The app uses a modern database to securely save your history and a calculation engine that analyzes your quiz scores in real-time to provide the most appropriate support resources.
